Hexo博客搭建指北

2022/11/26 教程 Hexo Github

搭这个博客花了不少时间，就想写一篇教程。

前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Markdown 语法的使用可以看 GitHub 官方文档：基本撰写和格式语法

在本地部署博客

在创建一个 github page 上的博客前，可以先在本地部署博客，再传到网络上。

1. 安装 Node.js 和 Git

下载网址 Node.js    Git - Downloads

完成后用 node -v 和 git --version 检查，有提示即安装正确

2. 安装 Hexo CLI

输入命令 npm i -g hexo， 用 hexo -v 检查安装

新建一个文件夹，作为博客目录，cd 进入文件夹，运行 hexo init & npm i

Hexo 官方文档：文档 | Hexo

3. Hexo 命令

生成静态文件    hexo generate    hexo g

清除静态文件    hexo clean    hexo cl

在本地运行        hexo server    hexo s

部署到网站        hexo deploy    hexo d

生成静态文件并部署到网站：hexo d -g 或 hexo g -d

创建新文章：hexo new <post-name> <post-name> 为文件名

4. 安装主题

在这里选择一个主题

在根目录下运行以下指令通过git把主题Clone到本地

<theme-dir> 为主题链接，<theme-name> 为主题名称

git clone <theme-dir>.git themes\<theme-name>

我使用的主题是 ParticleX

在博客根目录下的 _config.yml 中设置 theme 参数为你的主题名称，就可以切换主题，一般主题在 GitHub 项目页下都会有介绍和配置说明。

5. 创建关于页和分类

输入命令 hexo new page categories 创建 Categories 页面

输入命令 hexo new page tags 创建 Tags 页面

输入命令hexo new page about创建About关于页，再在md文件内添加内容

6.网站配置

打开博客根目录下的 _config.yml，这是一些比较重要的参数，其他可以保持默认

title: # 标题
subtitle: # 副标题
description: # 描述
keywords: # 关键字
author: # 作者
language: # 语言
timezone: # 时区
url: # 网址
root: # 根目录

更多配置可以参考Hexo 官方文档

将静态网页上传到Github Page

1. 新建Github Page

新建一个仓库，名称为<name>.github.io

name 是你的用户名

2. 配置 SSH 密钥

在命令行输入命令 ssh-keygen -t rsa -C <email>，获取 SSH 密钥，这个过程会提示你输入一些东西，一直回车就行了。Email 是你注册 Github 的邮箱

输入命令 notepad %USERPROFILE%\.ssh\id_rsa.pub，打开 SSH 密钥

打开你 GitHub 右上角的头像中的 Settings 设置，点击左边的 SSH and GPG keys，点击右上角的 New SSH key，将刚才打开的 SSH 密钥复制到 Key 中，点击 Add SSH key 添加密钥

设置好 SSH 密钥后用 ssh -T git@github.com 检测，如果出现 Hi, xxx! You've successfully authenticated 则配置成功

3. 部署到 GitHub Pages

因为默认分支以前是 master，现在是 main，所以如果不把分支名改为 master，就要将 _config.yml 的参数里的 branch: master 改为 branch: main

使用下面的命令初始配置

git config --global user.name "<name>" 
git config --global user.email "<email>"

打开你博客根目录的 _config.yml，设置参数

url: https://<name>.github.io/
# ...
deploy:
    type: git
    repo: git@github.com:<name>/<name>.github.io.git
    branch: master
    message:

安装部署插件    npm i -S hexo-deployer-git

在博客根目录下使用 hexo d -g 部署到 GitHub Pages
